EXECUTION CONTEXT

JS is a Synchronous Single Threaded language.

Everything in JS happens inside an execution context.

It's like as a big box whose have two components.

  • Memory execution / Variable environment: all variables and function stored here. Variable store here in key:value format.
  • Code/Thread of execution: here all code are executed. Execute one line at a time in a Synchronous way that means first thing first done then go to the next line. So JS is called Synchronous Single Threaded language.
